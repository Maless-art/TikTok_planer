MANUELITO STUDIO — VERSIÓN DE RESCATE

NO BORRA NI SOBRESCRIBE AUTOMÁTICAMENTE.

Qué hace:
1. Revisa todas las claves localStorage del mismo dominio.
2. Busca datos de las versiones V1, V2, Studio y otras claves compatibles.
3. Si encuentra una versión con más videos/series, la muestra temporalmente.
4. Solo escribe en la clave actual cuando presionas “Recuperar información”.
5. Antes de recuperar, descarga automáticamente un respaldo completo.
6. Incluye el logo definitivo en icon-192.png e icon-512.png.

MUY IMPORTANTE:
- Debe subirse al MISMO dominio donde se usaba la app anterior.
- No borres datos del navegador ni desinstales la PWA.
- Si no aparece el panel amarillo, usa “Descargar respaldo completo” desde la misma página y conserva el archivo.
- Si la clave original ya fue realmente sobrescrita y no existe otra copia, el navegador no ofrece una papelera para localStorage.
